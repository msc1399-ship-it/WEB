# Carnero Consultores Web

Landing independiente para presentar servicios, tarifas y formulario de contacto de Carnero Consultores.

## Estructura

- `index.html`: contenido y secciones de la web
- `styles.css`: estilos visuales y responsive
- `script.js`: menu movil, scroll suave y animaciones

## Como abrirla

Abre `index.html` directamente en el navegador para verla en local.

## Siguientes pasos recomendados

1. Sustituir telefono, email y datos legales por los reales.
2. Conectar el formulario a un destino real: Formspree, email, WhatsApp, HubSpot o backend propio.
3. Crear las paginas de aviso legal y politica de privacidad.
4. Publicarla en Netlify, Vercel o tu hosting habitual.
